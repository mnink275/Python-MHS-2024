# How to run (Linux)

### Set up environment
```
conda env create -f environment.yml -n YOUR_ENV_NAME
conda activate YOUR_ENV_NAME
```

### HW 1
```
cd hw1/task1/
python3 nl ./data/main.cpp
```

### HW 2
```
cd hw2/
docker compose up --build
```

### HW 3
```
cd hw3/
python3 matrix3.1.py
python3 matrix3.2.py
```
